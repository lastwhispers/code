# 类加载过程

加载（Loading）、验证（Verification）、准备（Preparation）、解析（Resolution）、初始化（Initialization）、使用（Useing）、卸载（Unloading）7个阶段。其中验证、准备和解析3个部分统称为连接（Linking）。

- 加载：类加载器，通过一个类的全限定名来获取此类的二进制字节流。
